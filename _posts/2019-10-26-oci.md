10月份回NYU招人，连续两天的车轮战面试还是让人略感疲惫。可尽管如此，我还是跟recruiter说这次的校招是“汗水与快乐并存，我非常享受”——80分矫揉造作，100分真情实感。

那么愉悦感从何而来？

其一，跟小朋友们对话让我感觉回到了过去。面试前一晚，我们专门在纽约办公室邀请候选人来meet up。在与之交流时，我能强烈地感受到小朋友们眼睛中锐利的光芒——那是一种对近在咫尺机会的狂热渴望。这种感觉，我曾经也有过。但今天是这么几年来第一回再次被它触动。

其二，新鲜的工作schedule。对比起平时email, code review, meetings, code etc.的单调生活，校招这两天的日程安排完全不同。尽管面试和写评定占了绝大多数时间，但聊天打趣、异于平日工作的0 context switching, 让我深深感觉仿佛在做另外一份工作，过另外一种人生。这种新鲜感也是工作多年以来未曾体会的。

飞回湾区的那一天，我非常地沮丧。低落到夜深人静还需要在沙发上发呆排遣——我到底想过什么样的生活，我到底怎么样才能快乐。生活中很多事情会让我忙起来——为老婆操心、为工作操心，这些都是我发自内心乐于去做的事。可是我真的不需要非常自私地为自己想想吗？我真的不渴望一种确确实实，完完全全几乎可以被称为为一己私欲的快乐吗? 

我原以为这样的事物或许对我来说并不存在，结果纽约的这几天我却品到了它些许滋味。可惜还没来得及好好回味，它便已烟消云散了。
